# ZÜMA STORE

Official online store of ZÜMA.

The Store is the commercial interface of the brand.

Its only responsibility is to present released objects and allow visitors to purchase them.

It does not explain the philosophy of the project.

It does not document the creative process.

For documentation, research and archives, see ZÜMA INDEX.

---

## Philosophy

The Store distributes.

The Index documents.

These two websites should never be merged.
